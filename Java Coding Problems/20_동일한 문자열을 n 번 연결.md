# 동일한 문자열을 n 번 연결

```java
public static String concatRepeat(String str, int n) {
  StringBuilder sb = new StringBuilder(str.length() * n);

  for (int i = 1; i <= n; i++) {
    sb.append(str);
  }

  return sb.toString();
}
```

## java 11

```java
String result = "hello".repeat(5);
```

String result = Stream.generate(() -> TEXT)
